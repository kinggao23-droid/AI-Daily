## 1. 今日要闻

### 🔥 要闻一: AWS Bedrock AgentCore 发布 Agent 自动优化能力 — 推荐 + 批量评估 + A/B 测试三件套上线

**平台**: AWS Bedrock AgentCore  |  **时间**: 2026 年 5 月初 (Preview)

**具体更新内容**:
- **Recommendations（推荐优化）**: AgentCore 自动分析生产环境中的 trace 和评估结果，生成针对特定工作负载优化的系统提示词（System Prompt）和工具描述（Tool Descriptions），无需人工逐条调优
- **Batch Evaluations（批量评估）**: 将推荐结果与预定义测试集进行批量对比验证，确保改动不会引入回归问题
- **A/B Testing（A/B 测试）**: 在生产流量上对推荐方案进行对照实验，系统自动计算统计显著性后才推荐推广
- **安全控制**: 所有优化推荐需人工审批后方可生效，不会自动修改生产配置
- **区域扩展**: AgentCore 于 5 月 1 日新增南美洲（圣保罗）区域支持

**影响分析**:
- **通俗解释**: 以前调优 AI Agent 像手动调收音机频率，现在 AWS 帮你自动扫频并推荐最佳频道，你只需要按"确认"
- **对竞品的压力**: Azure AI Foundry 和 Google Vertex AI 目前缺少类似的"闭环自动优化"能力，Agent 调优仍依赖人工 prompt engineering
- **对企业**: Agent 上线后的持续优化成本将显著降低，特别适合已在 AWS 上部署大量 Agent 的团队

**参考链接**:

- [AWS Blog: AgentCore 新功能详解](https://aws.amazon.com/blogs/machine-learning/get-to-your-first-working-agent-in-minutes-announcing-new-features-in-amazon-bedrock-agentcore/)

### 🔥 要闻二: GitHub Copilot 6 月 1 日全面转向按 Token 计费 — 开发者生态定价范式转变

**平台**: GitHub Copilot (Microsoft)  |  **时间**: 6 月 1 日生效，当前倒计时 29 天

**具体更新内容**:
- **计费模式**: 从"固定月费无限用"变为"月费 + Token 消耗额度"，超出部分按量计费
- **AI Credits 体系**: 1 AI Credit = $0.01 USD，每次调用根据模型和 Token 数量扣减 Credits
- **各档套餐保持原价但内涵变化**:

| 套餐 | 月费 | 包含 Credits | 超出后 |
|------|------|-------------|--------|
| Pro | $10/月 | 待公布 | 按 Token 计费 |
| Pro+ | $39/月 | 更多 | 按 Token 计费 |
| Business | $19/人/月 | 待公布 | 按 Token 计费 |
| Enterprise | $39/人/月 | 最多 | 按 Token 计费 |

- **重要豁免**: 代码补全（Code Completions）和 Next Edit 建议**不消耗** AI Credits，仍为无限使用
- **过渡优惠**: 现有 Business/Enterprise 客户在前 3 个月（6/1-9/1）享受更高的免费 Credits 额度

**影响分析**:
- **对开发者**: 高频使用 Copilot Chat 和 Agent 模式的开发者月度成本可能上升；仅使用代码补全的用户不受影响
- **对 LLMOps 平台**: 这是行业从"订阅制"向"按量计费"转型的风向标，预示着更多 AI 工具将采用类似模式
- **对企业采购**: 需重新评估 Copilot 实际使用成本，关注 Token 消耗的可预测性和预算控制

**参考链接**:
- [GitHub 官方博客: Usage-Based Billing](https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/)
- [GitHub Docs: 模型与定价](https://docs.github.com/en/copilot/reference/copilot-billing/models-and-pricing)
- [The Decoder 报道](https://the-decoder.com/github-copilot-switches-to-token-based-billing-in-june-2026/)

### 重点功能深度解读

**功能一: AWS AgentCore "Observe → Evaluate → Improve" 闭环**
- **通俗解释**: 就像电商平台的"千人千面推荐系统"，但推荐的是 Agent 的最佳配置——系统分析 Agent 在实际生产中的表现，自动找出哪些提示词和工具描述能让 Agent 表现更好，然后通过 A/B 测试验证后再上线
- **对产品经理的意义**: Agent 运维从"一次性部署"变为"持续自优化"，平台层需要提供类似的自动调优基础设施
- **落地建议**: 关注 AgentCore 优化功能 GA（正式发布）时间节点，评估是否将其纳入企业 Agent 运维标准流程

**功能二: Snowflake Cortex Code MCP Server + Claude Code 插件**
- **通俗解释**: Snowflake 把自己的 AI 编码助手变成了一个"万能插头"，可以插到 VS Code、Cursor、Claude Code 等任何主流 AI 编码工具中使用，而且在任何一个工具中创建的技能可以在其他工具中直接复用
- **对产品经理的意义**: 数据平台正在通过 MCP 协议渗透到开发者的 IDE 环境中，"数据 + AI 编码"的融合正在加速
- **落地建议**: 如果产品涉及数据分析场景，评估 Cortex Code MCP Server 是否可以为客户提供更好的数据开发体验

## 4. 新兴平台雷达

### 🆕 Claude Mythos Preview — Anthropic 面向网络安全的专用前沿模型

AWS Bedrock 于 4 月 7 日上线 Claude Mythos Preview（Project Glasswing 的一部分），这是 Anthropic 专为网络安全领域打造的模型，能力包括:
- 识别软件中的高级安全漏洞并演示可利用性
- 理解大型代码库并提供可操作的安全发现
- 比此前 AI 模型所需的人工指导更少

**访问限制**: 仅限邀请制，优先面向互联网关键基础设施企业和开源维护者，Anthropic 目前不计划公开发布。

**为什么值得关注**: 这是首次有主流 AI 厂商为单一行业（网络安全）发布专用前沿模型，可能预示着"行业专用大模型"将成为新的竞争维度。如果你的平台服务于金融、政务等高安全要求行业，应关注类似专用模型的出现。

## 5. 对产品经理的启示

### 启示一: Agent 运维正在从"手动调优"进入"自动闭环"时代

AWS AgentCore 的"推荐→评估→A/B 测试→上线"闭环标志着 Agent 运维工具的成熟度上了一个台阶。对产品经理的意义:
- **短期**: 如果你的产品包含 Agent 功能，评估是否可以接入 AgentCore 的优化能力来降低运维负担
- **中期**: 自建 LLMOps 平台应考虑将"Agent 自动调优"纳入路线图——这将从"Nice to have"变为"Must have"
- **行动项**: 调研 AgentCore 优化 API 的接入方式，评估是否可以将其包装为平台的增值服务

### 启示二: AI 工具计费正在从"订阅制"向"按量计费"全面转型

GitHub Copilot 的转型不是孤例——AWS Bedrock 按 Token 计费、DeepSeek 的缓存分级定价、阿里百炼的 Credits 套餐制，都指向同一个方向: 用户为实际消耗的算力付费，而非为使用权付费。对产品经理的意义:
- **成本可预测性**: 按量计费让大规模使用的成本变得不确定，企业需要更好的用量监控和预算管控工具
- **平台机会**: "AI 成本管理"（FinOps for AI）正在成为独立的产品赛道，可考虑在平台中内置用量预测和成本优化建议
- **行动项**: 如果你的产品采用订阅制，开始规划向混合计费模式（基础订阅 + 按量超额）的迁移路径

### 启示三: DeepSeek V4 Pro 促销窗口关闭后，重新评估模型选型

5 月 5 日促销结束后，V4 Pro 恢复标准价（输出 $3.48/M）将与 Qwen3.6-Plus（$3.00/M）处于同一价位区间。此时:
- **性能对比重新洗牌**: 在相近价格下，需要基于实际业务场景的基准测试结果来选择，而非仅看价格
- **缓存策略仍是关键**: 即使恢复标准价，DeepSeek 的缓存输入价格（$0.10/M）仍极具竞争力，高缓存命中率场景下 DeepSeek 依然有成本优势
- **行动项**: 在促销结束前，锁定关键业务场景的 DeepSeek 预充值额度；同时启动 Qwen3.6-Plus 的并行评测
